# DevMate AI

DevMate AI is a monorepo starter for building an AI-assisted developer workspace with a React frontend and an Express backend.

## Structure

- frontend/ - Vite + React application
  - src/components
  - src/pages
  - src/services
  - src/hooks
  - src/styles
- backend/ - Node.js + Express API
  - controllers/
  - routes/
  - services/
  - middlewares/
  - config/
  - utils/
